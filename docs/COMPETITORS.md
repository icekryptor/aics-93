### Паттерны топов, применимые к AICS-93

1. Продукт вместо услуги (Лебедев): фикс-цена + фикс-срок + scope + лимит слотов
   («беру N сайтов в месяц») + витрина работ/статистики продукта. Наш ответ на их боль:
   «2 итерации включены».
2. Лестница цен: дешёвый ИИ-вход (аудит/концепт) → экспресс-сайт 7–14 дней →
   дизайн-система/сервис. Нижний этаж маркетирует верхний.
3. Подписка отдельным продуктом («Очень»): «поддержка по подписке» с фикс-ставкой/мес.
4. ИИ-конвейер как продукт (Снэпбилд): показать пайплайн — таймлапс, дни, стоимость.
5. Метрика в заголовке кейса + голосом клиента: одна цифра + одна цитата.
6. Умный бриф + SLA ответа + «что будет дальше» — соло даёт SLA агрессивнее интегратора.
7. GEO-страница «Hey AI, learn about us» — копеечно, для ИИ-студии ещё и позиционирует.
8. Счётчики-доказательства на первом экране: запущено N, медианный срок X дней, PageSpeed 95+.
9. **Свободная поляна: ни один топ не публикует цену и срок на первом экране** — оффер
   «сайт за 7–14 дней от N ₽ с дедлайном в договоре» не конкурирует с ними, а закрывает
   аудиторию, которую они сознательно отсекают.

### Паттерны упаковки брендинга для AICS-93

1. **Ценовой разрыв, размеченный самой Repina**: «фрилансер 30–50 т.р.» ↔ «агентство от
   600 т.р.». Между ними пусто — AICS-93 встаёт на **150–300 т.р. / 10–14 дней**.
2. **Формула продуктовой карточки** (Repina+Depot): название → цена «от» → точный состав
   (8 пунктов) → срок в днях → что включено бесплатно → от чего растёт цена. Depot добавляет
   строки «разработка / финализация» — удобно для допродаж.
3. **Дизайн-система для МСБ не продаётся никем** (у Repina от 660 т.р., у ONY — этап,
   у Pinkman — кейс ВТБ). Наш оффер: **«фирстиль, который сразу живёт в коде»** — токены,
   UI-кит, компоненты для сайта на конвейере. Техпреимущество, которого нет ни у одного.
4. **Сроки этапов публично** (ENDY: 8 недель) — скопировать структуру, сжать в 4–5 раз:
   скорость бьёт всех шестерых одновременно.
5. **Входной продукт-диагностика**: ИИ-аудит бренда/сайта за фикс 15–30 т.р. / 3 дня
   (паттерн ONY «Дизайн-аудит», Depot «экспресс-аудит»).
6. **ИИ-нарратив уже легитимен** (Pinkman, ENDY, Mish, ONY) — оправдывать ИИ не нужно,
   отстраиваемся конкретикой: фикс-срок + фикс-цена + человек-владелец процесса.
7. **Таблица «3 пакета фирстиля» со сравнением состава на рынке отсутствует** — занять
   первым в сегменте МСБ.
8. Метрики в кейсах показывает только Pinkman — для соло без наград цифры+сроки — самый
   дешёвый способ выглядеть убедительнее агентств.
9. Лид-механики: личный контакт основателя (Pinkman) · шутливый CTA у брифа (Mish: «Бриф сам
   себя не заполнит») · поле «бюджет» как квалификация (Depot) · своя SEO-страница
   «сколько стоит фирстиль» с сегментацией рынка (Repina).

## 3. Паттерны рынка (гигиенический минимум — у всех)

1. Оффер «срок + под ключ» в первом экране; часто цена/срок прямо в title.
2. «Перед дизайном — анализ конкурентов и ЦА» — штамп сегмента (говорят одними словами).
3. Квиз/калькулятор + Telegram/WhatsApp как канал — у 5 из 7. Голая форма «имя+телефон» умерла.
4. Каноническая структура главной: оффер → преимущества → портфолио → тарифы → процесс →
   отзывы → FAQ → форма.
5. Тильда как субстрат у бутиков и соло — и одновременно их потолок по сложности.

## 4. Белые пятна рынка (никто не делает)

1. **Связка «быстрый сайт + фирстиль + веб-сервис + ИИ» в одном операторе.** Сайтоделы не трогают
   ИИ; ИИ-студии не делают сайты; m.ai заявляет всё, но ИИ там витрина.
2. **ИИ-конвейер как объяснение скорости и цены.** Никто не показывает, КАК нейросети ускоряют
   производство. Механика «один инженер + ИИ-пайплайн = скорость агентства за полцены» свободна.
3. **Инженерная эстетика.** У конкурентов — Тильда-стиль, агентский глянец или шаблон. Ниша
   «терминальность, схемы, живой код, артефакты процесса» пуста — это ровно наш стиль.
4. **Фикс-цена + фикс-срок + кейсы с цифрами одновременно.** Цены публичны у Landing.ru/Мегагрупп
   (без метрик), метрики у «Имя.»/aify (без цен). Комбинация свободна.
5. **Личный бренд инженера-оператора.** Единственный «я-оператор» — Коган (без ИИ и инженерности).
   Формат «build in public, разборы пайплайна» никем не занят.
6. **Дизайн-системы как продукт для МСБ.** У всех «фирстиль» = логотип-допуслуга (+60 т.р.),
   а не система.
7. **Живой продукт до сделки.** Ни у кого нет «прототип за 24 часа до оплаты» — при том что
   ИИ-конвейер делает это дёшево.

## 5. Карта отстройки AICS-93

### Уже уникально (усиливать и проговаривать)
- Позиция «индивидуальный сайт за 7–14 дней» — ниша между конвейером (3 дня, шаблон)
  и бутиком (30–45 дней) фактически пустует.
- Полный стек: сайт + фирстиль/дизайн-система + сервисы + ИИ — ни у кого из сегмента.
- Инженерная эстетика сайта — мгновенная визуальная отстройка от Тильда-однообразия.
- Квиз-механики уже на уровне лучших («Имя.», Komarovaeee) — есть и дровер, и инлайн.
- Направления-решения (товарка/услуги) с экономикой перехода — формата нет ни у кого.

### Дожать (кандидаты в план — детально в SITE-STRATEGY.md)
1. **Объяснить механику ИИ-конвейера** — блок/страница «как один инженер делает работу
   четверых»: пайплайн, harness, артефакты. Это же отстройка от нейро-конструкторов.
2. **Публичные пакеты с фикс-ценой и сроком** — примеры смет уже есть (Гант), нет цен.
3. **Кейсы с бизнес-метриками** — формат «Было → Стало → Результат» (paттерн aify),
   сейчас кейс 01 без цифр.
4. **Низкопороговый вход** — «напишите в TG „хочу структуру“» / бесплатный ИИ-аудит /
   прототип за 24 часа.
5. **Личный бренд оператора** — фото/история/процесс на видном месте (сейчас «оператор»
   абстрактен), связка с каналом @aics93.
6. **Регалии и соцдоказательства** — отзывы, логотипы клиентов, публикации (у нас ноль).
