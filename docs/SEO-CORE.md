# SEO-CORE — семантическое ядро aistov.space

Постраничная карта ключевых запросов: что продвигаем, на какой странице, куда вхождения.
Собрано 2026-07-18 по живой выдаче (SERP-анализ, подсказки, title/H1 конкурентов в топе).

⚠️ **Частотности — качественные оценки** (В — высокая, С — средняя, Н — низкая, ЛТ — лонгтейл),
выведены по косвенным признакам: плотность коммерческих страниц, присутствие агрегаторов,
ценовые якоря в title. **Перед закупкой трафика/контент-планом прогнать список через
Yandex Wordstat** (базовая + «!точная» частота, срез по региону).

## 0. Стратегия ядра — где мы реально можем выиграть

Выдача по «голове» кластеров (сайт под ключ, логотип, лендинг) забита агрегаторами
(kwork, profi.ru, youdo, Яндекс Услуги) и хостерами/конструкторами — лобовое SEO там нерентабельно.
Ядро строим вокруг **пяти недообслуженных интентов**, где конкуренция низкая и оффер AICS-93 уникален:

1. **«Сайт на ИИ-конвейере под ключ»** — стык «разработка сайтов» × «нейросети»: конструкторы
   предлагают DIY, студии не упоминают ИИ. Оффер «студийное качество за 7–14 дней благодаря
   ИИ-конвейеру» прямых конкурентов в выдаче не имеет.
2. **Дизайн-системы** — 3–4 профильные страницы на весь рунет; «дизайн-система бренда»,
   «заказать UI-кит» — почти чистые хвосты.
3. **Внедрение ИИ для МСБ** — кластер молодой, агрегаторов нет; весь топ по «нейросети для
   малого бизнеса» — DIY-обзоры, услуги «под ключ с адекватным чеком» нет.
4. **Селлеры маркетплейсов → свой сайт** — растущая информационная выдача (vc, biz360),
   почти нулевая коммерческая (один aspro). Захват через контент + посадочную.
5. **Отраслевые хвосты сферы услуг** — «сайт для ‹ниша› с записью — от N ₽» слабо заняты
   (кроме стоматологий); онлайн-запись сама по себе — SaaS-территория, заходить связкой.

Паттерн title коммерческого топа (использовать всюду): `‹Услуга› под ключ — от N ₽ / за M дней`.
Срок — наш главный дифференциатор, цена — второй.

## 4. Гигиена вхождений (правила)

- Один primary-интент = одна страница; не каннибализировать (товарка ≠ услуга «сайты»).
- Title: ключ + дифференциатор (срок/механика) + `| AICS-93`; ≤ 60 симв. до пайпа — по возможности.
- Description: 140–160 симв., цена-якорь или срок + CTA.
- H1 может быть офферным (не обязан дублировать title дословно), но содержит корень ключа.
- Синонимы кластера — в подзаголовках H2/H3 и FAQ (FAQ уже с JSON-LD — усиливать вопросами из «Люди ищут»).
- Метрики-крючки в подзаголовках работают в топе: «конверсия 4,7%», «окупаемость 2–4 месяца».
